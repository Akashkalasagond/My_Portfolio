Alzheimer Dementia Classification and Detection


In my Machine Learning project, I did Alzheimer's Dementia Classification. Labeled brain images undergo preprocessing, featuring normalization and resizing. Employing a Convolutional Neural Network (CNN) architecture enhances image recognition. Hyperparameter tuning optimizes learning rates for performance. The dataset is split for training and testing, preventing overfitting. Model evaluation includes metrics like Accuracy, Precision, and Recall. Post-training, the model is deployed on Streamlit for scalability, enabling it to analyze new images and provide diagnostic outcomes. This streamlined process also facilitates data collection, amplifying the project's impact.

Diabetes Prediction using MLP(ANN)


This project focuses on predicting the likelihood of diabetes in individuals based on relevant health features. The prediction model is built using a Multilayer Perceptron (MLP), a type of Artificial Neural Network (ANN). By leveraging machine learning techniques, this project aims to contribute to early diagnosis and proactive management of diabetes.
